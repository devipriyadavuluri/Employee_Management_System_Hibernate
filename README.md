# Employee Management System using Hibernate

This is a Java-based Employee Management System developed using Hibernate ORM.

Features:
- Add new employee
- Update employee details
- Delete employee records
- View employee information

Technologies Used:
- Java
- Hibernate
- MySQL
- Maven
- Git & GitHub

Concepts Used:
- ORM (Hibernate)
- CRUD operations
- DAO design pattern
- Exception handling

Purpose:
This project was created to understand Hibernate and database interaction using ORM.
